# AGENTS.md

This scoped guidance applies only to child-agent fixture examples in this
directory.

Fixture reports are examples, not live audit evidence. Do not treat them as
authorization for edits, commits, pushes, installs, indexing, exports, releases,
publication, provenance, or AGENTS.md changes.

When updating these examples, preserve the root `AGENTS.md` hierarchy and the
rule that child-agent findings become implementation work only after the main
`/implementaudit` agent normalizes them into a ledger.
