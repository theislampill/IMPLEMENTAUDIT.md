## 9. Repairs made

- **F1 (P0):** `tests/custody-append.test.sh` — replaced the
  `case "${activegraph_cmd[0]}" in *.exe|/mnt/*)` gating (which a bare
  `activegraph` shim bypasses) with a `need_win_url_path` flag computed from the
  **shell environment** (`uname -s` matching `MINGW*|MSYS*|CYGWIN*`) as well as
  the resolved command name, applied to both the `inspect` and `export-trace`
  URL paths. Additive and platform-safe: a no-op on plain Linux; preserves the
  prior `.exe`/`/mnt` triggers. Shipped `custody-append.sh` was **not** changed
  (its write path is a standalone arg that MSYS auto-converts; it has no bug).
- **F1 anti-repeat rule:** added `V0300-MSYS-URL-PATH-NORMALIZATION` to AGENTS.md
  (durable, repo-specific, would have prevented this finding).
- **F2:** no patch (by-design; documented above).
- **F3:** reconciled at release time only (see §13).
