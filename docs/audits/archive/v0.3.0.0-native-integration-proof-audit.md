are available through native IMPLEMENTAUDIT runtime routes: audit-object
lifecycle, owner/source routing, DMAIC/DMADV, Jidoka/Andon, Smoke A/B, Plan

This proof audit was read-only except for this document. No runtime files were
patched. No commit, push, tag, release, publication, provenance, issue creation,
donor mutation, hidden install, or sidecar export was performed.
